# Data Cleaning and Preprocessing Task

## 📌 Objective

The goal of this project is to clean and preprocess a raw dataset by handling:

* Missing values
* Duplicate records
* Inconsistent formatting
* Incorrect data types
* Standardized column names

---

## 🛠 Tools Used

* Microsoft Excel
* Python (Optional)
* Pandas Library

---

## 📂 Dataset Information

The dataset contains raw data with inconsistencies and missing values that were cleaned for better analysis and modeling.

---

## ✅ Data Cleaning Steps Performed

### 1. Handling Missing Values

* Identified null/blank values
* Removed or filled missing records where necessary

### 2. Removing Duplicates

* Checked duplicate rows
* Removed repeated records

### 3. Standardizing Data

* Fixed inconsistent text formats
* Standardized column names
* Corrected spelling inconsistencies

### 4. Date Formatting

* Converted dates into a consistent format

### 5. Data Type Corrections

* Ensured numeric columns were properly formatted
* Checked date and text columns

---

## 📊 Final Output

A cleaned and structured dataset ready for analysis and visualization.
